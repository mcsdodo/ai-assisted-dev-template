# Record Decision

Document an architectural (ADR) or business logic (BIZ) decision in `DECISIONS.md`.

**Quick reference:**
- **ADR**: Framework choices, design patterns, technical trade-offs
- **BIZ**: Calculations, validation rules, business logic

## Usage

This command invokes the decision skill. See `.claude/skills/decision-skill/SKILL.md` for the full workflow and templates.

## Quick Steps

1. Determine type (ADR or BIZ)
2. Find next number in `DECISIONS.md`
3. Add entry at TOP (newest first)
4. Commit the decision
